# 学习资料自动化预处理工具

这是一个面向日常学习资料整理场景的 Python 小工具，适合用来批量处理图片和 PDF 文件。  
它把常见操作整合成了一个交互式终端菜单，不需要记命令参数，按照屏幕提示一步步输入路径和选项即可完成处理。

## 主要功能

程序启动后会显示编号菜单，当前支持以下功能：

1. 批量转换图片为 WebP
将文件夹中的 PNG、JPG、JPEG、TIFF、BMP、GIF 等图片批量转换为 WebP 格式，方便压缩体积、节省存储空间。

2. 批量增强图片（OCR 预处理）
对图片进行灰度化、去噪、对比度增强、二值化等处理，帮助提升后续 OCR 识别效果。

3. 拆分长 PDF
将页数较多的 PDF 按指定页数拆分为多个小 PDF，便于上传、分享或分段处理。

4. PDF 转图片（并可选拼接长图）
将 PDF 每一页导出为图片，并可按页数分组拼接成长图。  
为了避免单张图片过长导致内存占用过高，程序默认最多每 10 页拼接成一张长图。

## 环境安装

### 1. 安装 Python

建议使用 Python 3.10 及以上版本。

你可以先在终端中输入下面的命令，确认 Python 是否已经安装：

```bash
python --version
```

### 2. 安装项目依赖

进入项目目录后，执行下面的命令安装依赖：

```bash
pip install -r requirements.txt
```

如果你的电脑上同时存在多个 Python 环境，也可以使用：

```bash
python -m pip install -r requirements.txt
```

## 使用说明

### 1. 运行脚本

在项目目录中打开终端，执行：

```bash
python preprocess_materials.py
```

### 2. 根据菜单选择功能

程序启动后会显示类似下面的菜单：

```text
1. 批量转换图片为 WebP
2. 批量增强图片 (OCR 预处理)
3. 拆分长 PDF
4. PDF 转图片 (并可选拼接长图)
0. 退出
```

你只需要输入对应的数字，然后按提示继续操作即可。

例如：

- 选择 `1` 后，程序会提示你输入图片文件夹路径、输出目录和 WebP 质量。
- 选择 `3` 后，程序会提示你输入 PDF 文件路径、输出目录以及每个拆分文件最多多少页。
- 选择 `4` 后，程序会提示你输入 PDF 路径、导出图片目录、DPI，以及是否需要拼接长图。

### 3. 查看处理结果

如果你直接按回车使用默认输出目录，程序通常会在原文件旁边自动生成以下目录之一：

- `webp_output/`
- `ocr_enhanced/`
- `pdf_split/`
- `pdf_images/`

处理过程中会显示进度条，方便你看到当前处理进展；如果个别文件出错，程序也会保留错误提示，便于排查问题。

## 适用场景

这个工具比较适合以下情况：

- 想把扫描图片批量压缩成 WebP
- 想先增强图片，再拿去做 OCR 识别
- 想把一个大 PDF 拆成多个小文件
- 想把 PDF 转成图片，或者整理成长图用于阅读、做笔记

## 说明

- 本项目是一个本地处理工具，不依赖在线服务。
- 建议先用少量文件测试一次，确认效果符合预期后再批量处理。
- `.gitignore` 已默认忽略处理结果目录和常见原始素材文件，避免把大体积资料误传到仓库中。
