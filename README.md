# 学习资料自动化预处理工具

一个专注提升 AI 读取效率的学习资料预处理工具，帮助你在把图片和 PDF 交给大模型、OCR 或知识库之前，先完成体积压缩、图像增强与结构整理，减少无效信息干扰，提升后续识别与阅读效果。

## 功能列表

### 1. 图片转 WebP

批量将常见图片格式转换为 WebP，在尽量保持可读性的前提下显著节省体积，便于归档、上传和喂给 AI 工具处理。

### 2. OCR 图像增强

对图片进行灰度化、降噪、对比度增强、自适应二值化等预处理，提高 OCR 识别率，尤其适合扫描件、截图和拍照资料。

### 3. PDF 智能拆分与转图

- 按页数自动拆分长 PDF，方便分段处理与上传。
- 将 PDF 页面导出为图片。
- 支持按批次拼接成长图，适合连续阅读与长文档整理。

## 安装与运行

建议使用 Python 3.10 及以上版本。

安装依赖：

```bash
pip install -r requirements.txt
```

运行脚本：

```bash
python preprocess_materials.py
```

程序启动后会进入交互式终端菜单，你可以根据提示选择以下功能：

```text
1. 批量转换图片为 WebP
2. 批量增强图片 (OCR 预处理)
3. 拆分长 PDF
4. PDF 转图片 (并可选拼接长图)
0. 退出
```

## 项目亮点

- 面向 AI 读取场景设计，优先解决体积、清晰度和结构化问题。
- 交互式终端菜单降低使用门槛，不需要记忆复杂命令参数。
- 内置进度条与日志输出，方便观察处理进度和排查异常。
